# DWM config, made for My devices PC and ThinkPad x220

This build uses:
* dwm - window manager
    * fullgaps
    * systray
    * attachasideandbelow
* dwmblocks - same as slstatus but better
* dmenu - application menu
* kitty - terminal emulator
* i3lock - lockscreen
* mpv - media player
* picom - fancy graphics
* feh - wallpaper
